# AGENTS.MD

Este archivo describe las configuraciones y los archivos de contexto disponibles para los agentes de IA que trabajan con este proyecto.

## Agentes configurados

### Claude
- Carpeta: `.claude/`
- Archivo principal de instrucciones: `.claude/AGENT.md`
- Archivos de contexto adicionales:
  - `.claude/README.md`
  - `.claude/PROJECT_CONTEXT.md`
  - `.claude/TASKS.md`

## Propósito

Estos archivos proporcionan contexto del proyecto, instrucciones de uso y un mapa básico de los componentes principales para que el agente de IA pueda trabajar con el código de forma segura y efectiva.

## Uso

1. Consultar `.claude/AGENT.md` para ver las directrices del asistente.
2. Revisar `.claude/PROJECT_CONTEXT.md` para conocer la estructura del proyecto y los archivos clave.
3. Utilizar `.claude/TASKS.md` para ver los tipos de tareas esperadas.

## Notas

- El proyecto es una aplicación de nómina construida con Vite y React.
- Todos los agentes deben respetar la intención del usuario y trabajar en español.
