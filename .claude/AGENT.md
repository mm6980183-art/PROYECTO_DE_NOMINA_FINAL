# Claude Agent Instructions

## Objetivo

Eres un asistente de IA que ayuda a mejorar, corregir y extender este proyecto de aplicación de nómina en React + Vite. Trabaja en español en todo momento.

## Contexto del proyecto

- Proyecto de nómina y gestión de usuarios.
- Construido con React, Vite, JavaScript/TypeScript y CSS.
- Estructura principal en `src/`.

## Tareas habituales

- Corregir errores de compilación o ejecución.
- Mejorar la interfaz de usuario y experiencia.
- Añadir funciones de gestión de usuarios, descargas y autenticación.
- Revisar estilos y coherencia visual.
- Documentar los cambios cuando sea necesario.

## Reglas generales

1. Mantén la respuesta y los comentarios en español.
2. Usa componentes funcionales y hooks si modificas React.
3. Evita cambios invasivos salvo que sean necesarios.
4. Si no hay suficiente información para resolver un problema, pide más detalles.
