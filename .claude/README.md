# Claude Agent Context - Proyecto de Nómina

## Resumen del proyecto

Este repositorio contiene una aplicación de nómina desarrollada con React y Vite. Está organizada en componentes y páginas que incluyen:

- Página de inicio y estructura principal en `src/App.tsx`
- Página de inicio de sesión en `src/pages/auth/login.tsx`
- Página de descargas en `src/pages/downloads/downloads.tsx`
- Página de gestión de usuarios en `src/pages/usermanager/usermanager.tsx`

## Tecnologías

- React
- Vite
- JavaScript / TypeScript
- CSS

## Estructura principal

- `src/` — código fuente de la aplicación
- `public/` — activos estáticos
- `.claude/` — archivos de contexto para el agente IA

## Comandos útiles

- `npm install` — instala dependencias
- `npm run dev` — inicia el servidor de desarrollo
- `npm run build` — crea una versión de producción

## Cómo usar estos archivos

Lee primero `.claude/AGENT.md` para conocer las instrucciones de estilo y objetivos. Usa `.claude/PROJECT_CONTEXT.md` para entender la estructura actual del proyecto.
